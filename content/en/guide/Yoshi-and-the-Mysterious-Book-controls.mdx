### Default Control Layout

The core actions in *Yoshi and the Mysterious Book* are distributed across the Nintendo Switch 2's Joy-Cons or Pro Controller. While specific button assignments can be customized, understanding the default setup is crucial for your initial journey.

| Action | Default Button (Joy-Con R) | Default Button (Pro Controller) | Description |
|---|---|---|---|
| Move | Left Stick | Left Stick | Controls Yoshi's movement. |
| Jump / Flutter Jump | B / A | B / A | Basic jump; hold for flutter jump. |
| **Egg Throw** | R | R | Launches an egg. (Note: This is a change from previous titles.) |
| Tail Spin | Y | Y | A new move for interacting with the environment. |
| Gobble / Lick | X | X | Uses Yoshi's tongue to eat enemies or objects. |
| Ground Pound | ZL / ZR | ZL / ZR | Smashes downwards, useful for breaking objects or interacting with creatures. |
| Pause Menu | + | + | Accesses game menu, settings, and map. |
| Map / Hints | - | - | Opens the map and view discovery hints. |

<Section icon={Swords} title="Mastering Egg-Throwing Mechanics">
One of the most iconic abilities in any Yoshi game is the egg throw, and *Yoshi and the Mysterious Book* introduces a slight, yet impactful, change to its execution. Unlike some previous titles where the Y button might have been used, here, the **egg throwing button is mapped to R**. This might take some getting used to for seasoned Yoshi players.

<Callout type="warning" title="Adjusting to New Egg Throw">
Many players, especially those familiar with older Yoshi games, might instinctively reach for the Y button for egg throwing. Be prepared for a brief adjustment period as you retrain your muscle memory to use the R button for launching eggs.
</Callout>

The game's focus on exploration and experimenting with creatures means precise egg throws are often necessary. Whether you're aiming at a distant target while bouncing or trying to hit a specific environmental puzzle, the R button's placement on the top-right of the Joy-Con or Pro Controller can feel a bit challenging initially. However, the game also offers the Y button for certain egg-throwing situations, providing a bit of flexibility.

### Egg Throwing Techniques

| Technique | Description | Use Case |
|---|---|---|
| **Standard Throw** | Press and release R to throw an egg in a straight line. | Hitting enemies or objects directly in front of Yoshi. |
| **Aimed Throw** | Hold R to bring up an aiming reticle, then release to throw. | Precision shots, hitting targets at an angle or distance. |
| **Bouncing Throw** | Throw an egg while bouncing on a spring or creature. | Reaching high or hidden objects, solving vertical puzzles. |
| **Y-Button Throw** | In specific contexts, the Y button can also initiate an egg throw. | For "important situations" as described by some players, though specific scenarios are still being discovered. |

</Section>
